## Setup (local)
1. Open project folder:
   cd "e:\VS CODE\Projects\Web Dev Projects\Food Delivery Website"
2. Install dependencies:
   npm install
3. Start dev server:
   npm run dev
   (or `npm start` / `npm run serve` depending on project)
